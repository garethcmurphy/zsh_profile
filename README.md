## Installation  

1. Clone the repository:  
git clone https://github.com/your-username/zsh-profile.git  
cd zsh-profile  

2. Backup your existing Zsh profile:  
mv ~/.zshrc ~/.zshrc.bak  

3. Copy the provided Zsh profile to your home directory:  
cp zshrc ~/.zshrc  

4. Reload the Zsh profile:  
source ~/.zshrc
